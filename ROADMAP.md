# Banco de Dados

Tabela principal:

propostas

Nunca criar outra tabela para propostas.

Sempre utilizar a estrutura existente.

Antes de alterar qualquer coluna:

1. Ler a estrutura atual do banco
2. Verificar os tipos
3. Utilizar apenas colunas existentes

Nunca utilizar nomes antigos.

Exemplo:

❌ criadoEm

✅ created_at

7. Sempre fazer commit organizado.
